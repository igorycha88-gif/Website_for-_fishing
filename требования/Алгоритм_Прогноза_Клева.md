## 1. Научная основа (Solunar Theory)

### 1.1. Что такое Solunar Theory?

**Solunar Theory** (от лат. Sol = Солнце + Luna = Луна) — теория, разработанная Джоном Олденом Найтом в 1926 году, основанная на наблюдениях рыболовов и охотников за столетия.

**Основной принцип**: Рыба и животные более активны в определенные периоды, связанные с положением Солнца и Луны.

**Ключевые периоды**:
- **Major Periods** — когда Луна непосредственно над головой или под горизонтом (максимальная активность ~2 часа)
- **Minor Periods** — когда Луна восходит или заходит (умеренная активность ~1 час)

### 1.2. Научные исследования

Исследования подтверждают влияние следующих факторов:
1. **Фаза луны** — гравитационное влияние на водоемы
2. **Атмосферное давление** — влияет на плавательный пузырь рыбы
3. **Температура воды** — влияет на метаболизм рыбы
4. **Освещенность** — влияет на кормовое поведение

## 2. Факторы, влияющие на клев

### 2.1. Атмосферное давление ⭐⭐⭐⭐⭐ (КРИТИЧНО)

**Научное обоснование**:
- Рыбы имеют плавательный пузырь, который чувствителен к изменениям давления
- При падении давления рыба ощущает дискомфорт и меньше кормится
- При стабильном/растущем давлении — активность повышается

**Оптимальные значения**:
| Давление (мм рт.ст.) | Оценка | Поведение рыбы |
|---------------------|--------|----------------|
| 760-770 | Отлично (100%) | Максимальная активность |
| 755-760 | Хорошо (85%) | Активный клев |
| 750-755 | Умеренно (70%) | Нормальный клев |
| 745-750 | Слабо (50%) | Пассивна |
| < 745 или > 775 | Плохо (20%) | Рыба неактивна |

**Важнее тренд, чем абсолютное значение**:
- 📈 Растущее давление = клев улучшается
- ➡️ Стабильное = нормальный клев
- 📉 Падающее = клев ухудшается

```

Давление вчера: 755 мм
Тренд: +3 мм (рост) → Бонус +15%
```

### 2.2. Температура воздуха/воды ⭐⭐⭐⭐⭐ (КРИТИЧНО)

**Научное обоснование**:
- Рыба — хладнокровные существа, метаболизм зависит от температуры
- У каждого вида есть оптимальный температурный диапазон
- Слишком холодно = замедленный метаболизм
- Слишком жарко = дефицит кислорода, рыба уходит на глубину

**Оптимальные диапазоны по видам**:

| Рыба | Оптимум (°C) | Диапазон (°C) | Зимой |
|------|--------------|---------------|-------|
| Щука | 8-18 | 4-22 | ✅ Активна |
| Судак | 10-20 | 8-25 | ❌ Пассивен |
| Окунь | 12-20 | 6-25 | ✅ Активен |
| Карп | 18-25 | 15-28 | ❌ Спит |
| Карась | 15-25 | 10-28 | ✅ Активен |
| Лещ | 12-20 | 8-24 | ❌ Пассивен |
| Плотва | 8-18 | 4-22 | ✅ Активна |
| Форель | 8-16 | 6-18 | ❌ |
| Налим | 0-8 | -2-12 | ✅ Активен |

**Алгоритм расчета**:
```python
def temperature_score(actual_temp, fish):
    optimal_min = fish.optimal_temp_min
    optimal_max = fish.optimal_temp_max
    range_size = optimal_max - optimal_min
    
    if optimal_min <= actual_temp <= optimal_max:
        # В оптимуме — 100%
        return 100
    
    deviation = 0
    if actual_temp < optimal_min:
        deviation = optimal_min - actual_temp
    else:
        deviation = actual_temp - optimal_max
    
    # Линейное убывание: за каждые 5°C отклонения -25%
    penalty = (deviation / 5) * 25
    return max(0, 100 - penalty)
```

### 2.3. Ветер ⭐⭐⭐⭐ (ВАЖНО)

**Научное обоснование**:
- Сильный ветер мешает забросу и проводке
- Волнение воды насыщает её кислородом (хорошо)
- Ветер сдувает насекомых — рыба подходит к поверхности

**Правила**:
| Скорость (м/с) | Оценка | Влияние |
|----------------|--------|---------|
| 0-3 | Отлично (100%) | Идеальные условия |
| 3-5 | Хорошо (85%) | Комфортно для рыбалки |
| 5-8 | Умеренно (65%) | Трудно забрасывать |
| 8-12 | Слабо (40%) | Очень трудно |
| > 12 | Плохо (10%) | Рыбалка невозможна |

**Направление ветра (народные приметы)**:
- Южный/Западный = тепло, клев улучшается
- Северный/Восточный = холодает, клев ухудшается
- **Важно**: Ветер должен дуть в сторону берега, где рыба кормится

```python
def wind_score(speed, direction, fish):
    base_score = 100 - (speed / fish.max_wind_speed) * 50
    
    # Бонус за направление
    if direction in ['S', 'SW', 'W']:
        base_score += 10
    elif direction in ['N', 'NE', 'E']:
        base_score -= 10
    
    return max(0, min(100, base_score))
```

### 2.4. Фаза Луны ⭐⭐⭐ (УМЕРЕННО)

**Научное обоснование (Solunar Theory)**:
- Гравитация Луны влияет на водоемы (приливы)
- В полнолуние и новолуние — максимальная активность
- В четверть луны — минимальная активность

**Solunar Periods**:
| Период | Фаза | Активность | Длительность |
|--------|------|------------|--------------|
| Major | Полнолуние | Максимальная | ~2 часа |
| Major | Новолуние | Максимальная | ~2 часа |
| Minor | Растущая/Убывающая | Умеренная | ~1 час |
| Слабый | Четверть | Минимальная | — |

**Фаза луны (0-1)**:
- 0.0 = Новолуние → Major Period
- 0.25 = Первая четверть → Слабый
- 0.5 = Полнолуние → Major Period  
- 0.75 = Последняя четверть → Слабый
- 1.0 = Новолуние → Major Period

```python
def moon_score(moon_phase, fish):
    sensitivity = fish.moon_sensitivity  # 0.0-1.0
    
    # Расстояние до экстремумов (новолуние/полнолуние)
    distances = [
        abs(moon_phase - 0.0),   # Новолуние
        abs(moon_phase - 0.5),   # Полнолуние
        abs(moon_phase - 1.0),   # Новолуние (след.)
    ]
    min_distance = min(distances)
    
    # Ближе к экстремуму = выше балл
    # 0 = 100 баллов, 0.25 = 0 баллов
    base_score = 100 - (min_distance * 400)
    
    # Применяем чувствительность вида
    return 50 + (base_score - 50) * sensitivity
```

### 2.5. Осадки ⭐⭐⭐ (УМЕРЕННО)

**Научное обоснование**:
- Небольшой дождь — хорошо (насыщает воду кислородом)
- Сильный дождь — плохо (мутная вода, давление падает)
- Гроза — рыба уходит на глубину

**Шкала**:
| Осадки | Оценка | Влияние |
|--------|--------|---------|
| Ясно | 80% | Хорошо |
| Облачно | 90% | Отлично (комфортно рыбе) |
| Небольшой дождь | 75% | Хорошо для хищника |
| Дождь | 50% | Умеренно |
| Ливень | 20% | Плохо |
| Гроза | 10% | Опасно |

### 2.6. Время суток ⭐⭐⭐⭐⭐ (КРИТИЧНО)

**Научное обоснование**:
- Суточные ритмы кормления
- Рассвет/Закат = пики активности
- Полдень = часто затишье (особенно летом)

**Типичные периоды активности**:

|-------|--------|------------|

| 09:00-12:00 | Первая половина дня | Умеренная |

| 15:00-18:00 | Вторая половина дня | Умеренная |
| 18:00-21:00 | Вечерний пик | МАКСИМАЛЬНАЯ |
| 21:00-24:00 | Поздний вечер | Умеренная |
| 00:00-04:00 | Ночь | Зависит от вида |

### 3.3. Полный алгоритм (Python)

```python
from dataclasses import dataclass
from typing import Optional
from datetime import datetime, time
from enum import Enum

class TimeOfDay(Enum):
    MORNING = "morning"    # 06:00-10:00
    DAY = "day"           # 10:00-17:00
    EVENING = "evening"   # 17:00-21:00
    NIGHT = "night"       # 21:00-06:00

@dataclass
class WeatherConditions:
    temperature: float           # °C
    pressure: float              # мм рт.ст.
    pressure_trend: float        # изменение за 24ч
    wind_speed: float            # м/с
    wind_direction: str          # N/S/E/W/NE/SE/SW/NW
    cloudiness: int              # % (0-100)
    precipitation_mm: float      # мм осадков
    precipitation_type: str      # none/rain/snow/storm
    moon_phase: float            # 0.0-1.0
    sunrise: time
    sunset: time

@dataclass
class FishSettings:
    fish_name: str
    optimal_temp_min: float
    optimal_temp_max: float
    optimal_pressure_min: int
    optimal_pressure_max: int
    max_wind_speed: float
    prefer_morning: bool
    prefer_evening: bool
    prefer_overcast: bool
    moon_sensitivity: float      # 0.0-1.0
    active_in_winter: bool

def get_time_of_day(current_hour: int, sunrise: time, sunset: time) -> TimeOfDay:
    if 6 <= current_hour < 10:
        return TimeOfDay.MORNING
    elif 10 <= current_hour < 17:
        return TimeOfDay.DAY
    elif 17 <= current_hour < 21:
        return TimeOfDay.EVENING
    else:
        return TimeOfDay.NIGHT

def calculate_temperature_score(weather: WeatherConditions, fish: FishSettings) -> float:
    temp = weather.temperature
    opt_min = fish.optimal_temp_min
    opt_max = fish.optimal_temp_max
    
    if opt_min <= temp <= opt_max:
        return 100.0
    
    deviation = min(abs(temp - opt_min), abs(temp - opt_max))
    penalty = (deviation / 5.0) * 25
    return max(0.0, 100.0 - penalty)

def calculate_pressure_score(weather: WeatherConditions, fish: FishSettings) -> float:
    pressure = weather.pressure
    trend = weather.pressure_trend
    opt_min = fish.optimal_pressure_min
    opt_max = fish.optimal_pressure_max
    
    # Базовый балл по абсолютному значению
    if opt_min <= pressure <= opt_max:
        base_score = 100.0
    else:
        deviation = min(abs(pressure - opt_min), abs(pressure - opt_max))
        base_score = max(0.0, 100.0 - deviation * 3)
    
    # Бонус/штраф за тренд
    if trend > 3:          # Быстрый рост
        base_score += 15
    elif trend > 0:        # Медленный рост
        base_score += 8
    elif trend < -3:       # Быстрое падение
        base_score -= 20
    elif trend < 0:        # Медленное падение
        base_score -= 10
    
    return max(0.0, min(100.0, base_score))

def calculate_wind_score(weather: WeatherConditions, fish: FishSettings) -> float:
    speed = weather.wind_speed
    direction = weather.wind_direction
    max_wind = fish.max_wind_speed
    
    if speed <= max_wind * 0.5:
        base_score = 100.0
    elif speed <= max_wind:
        base_score = 100.0 - (speed / max_wind) * 30
    else:
        excess = speed - max_wind
        base_score = max(0.0, 70.0 - excess * 10)
    
    # Бонус за направление (теплые ветра)
    if direction in ['S', 'SW', 'W']:
        base_score += 10
    elif direction in ['N', 'NE', 'E']:
        base_score -= 10
    
    return max(0.0, min(100.0, base_score))

def calculate_moon_score(weather: WeatherConditions, fish: FishSettings) -> float:
    phase = weather.moon_phase
    sensitivity = fish.moon_sensitivity
    
    # Ближе к 0, 0.5 или 1 = лучше
    distances = [
        abs(phase - 0.0),
        abs(phase - 0.5),
        abs(phase - 1.0),
    ]
    min_distance = min(distances)
    
    # 0.0 = 100, 0.25 = 0
    raw_score = 100.0 - min_distance * 400
    raw_score = max(0.0, raw_score)
    
    # Смешиваем с базовым 50%
    return 50.0 + (raw_score - 50.0) * sensitivity

def calculate_precipitation_score(weather: WeatherConditions) -> float:
    precip = weather.precipitation_mm
    precip_type = weather.precipitation_type
    
    if precip_type == 'storm':
        return 10.0
    elif precip_type == 'snow':
        return 60.0
    elif precip > 10:  # Сильный дождь
        return 30.0
    elif precip > 5:   # Дождь
        return 50.0
    elif precip > 0:   # Небольшой дождь
        return 75.0
    elif weather.cloudiness > 70:  # Облачно
        return 90.0
    else:  # Ясно
        return 80.0

def calculate_time_score(time_of_day: TimeOfDay, fish: FishSettings) -> float:
    scores = {
        TimeOfDay.MORNING: 100.0 if fish.prefer_morning else 60.0,
        TimeOfDay.DAY: 65.0,
        TimeOfDay.EVENING: 100.0 if fish.prefer_evening else 60.0,
        TimeOfDay.NIGHT: 40.0,
    }
    return scores[time_of_day]

def get_season_multiplier(fish: FishSettings, month: int) -> float:
    if month in [12, 1, 2]:  # Зима
        return 1.2 if fish.active_in_winter else 0.3
    elif month in [3, 4, 5]:  # Весна
        return 1.0
    elif month in [6, 7, 8]:  # Лето
        return 1.0
    else:  # Осень
        return 1.0

def calculate_bite_score(
    weather: WeatherConditions,
    fish: FishSettings,
    hour: int,
    month: int
) -> dict:
    time_of_day = get_time_of_day(hour, weather.sunrise, weather.sunset)
    
    temp_score = calculate_temperature_score(weather, fish)
    pressure_score = calculate_pressure_score(weather, fish)
    wind_score = calculate_wind_score(weather, fish)
    moon_score = calculate_moon_score(weather, fish)
    precip_score = calculate_precipitation_score(weather)
    time_score = calculate_time_score(time_of_day, fish)
    
    # Взвешенная сумма
    bite_score = (
        temp_score * 0.25 +
        pressure_score * 0.25 +
        time_score * 0.20 +
        wind_score * 0.15 +
        moon_score * 0.10 +
        precip_score * 0.05
    )
    
    # Сезонный множитель
    season_mult = get_season_multiplier(fish, month)
    bite_score *= season_mult
    
    # Ограничение 0-100
    bite_score = max(0, min(100, bite_score))
    
    return {
        'bite_score': round(bite_score, 1),
        'time_of_day': time_of_day.value,
        'components': {
            'temperature': round(temp_score, 1),
            'pressure': round(pressure_score, 1),
            'wind': round(wind_score, 1),
            'moon': round(moon_score, 1),
            'precipitation': round(precip_score, 1),
            'time_of_day': round(time_score, 1),
        },
        'season_multiplier': season_mult,
    }

def generate_recommendation(score: float, weather: WeatherConditions, fish: FishSettings) -> str:
    recommendations = []
    
    # Базовая оценка
    if score >= 80:
        base = "Отличный клев! Идеальное время для рыбалки."
    elif score >= 65:
        base = "Хороший клев. Рекомендуется выйти на воду."
    elif score >= 50:
        base = "Умеренный клев. Можно рассчитывать на улов."
    elif score >= 35:
        base = "Слабый клев. Потребуется терпение и правильная тактика."
    else:
        base = "Клев маловероятен. Лучше перенести рыбалку."
    
    recommendations.append(base)
    
    # Специфические советы
    if weather.wind_speed > 6:
        recommendations.append("Ветреная погода — используйте тяжелые приманки.")
    
    if weather.pressure_trend < -3:
        recommendations.append("Давление падает — рыба пассивна.")
    elif weather.pressure_trend > 3:
        recommendations.append("Давление растет — клев должен улучшаться.")
    
    if weather.temperature < fish.optimal_temp_min:
        recommendations.append("Холодная вода — проводка должна быть медленной.")
    elif weather.temperature > fish.optimal_temp_max:
        recommendations.append("Теплая вода — рыба держится на глубине.")
    
    if weather.moon_phase < 0.1 or weather.moon_phase > 0.9:
        recommendations.append("Новолуние — хороший период по Solunar теории.")
    elif 0.4 < weather.moon_phase < 0.6:
        recommendations.append("Полнолуние — пиковая активность.")
    
    return " ".join(recommendations)
```
