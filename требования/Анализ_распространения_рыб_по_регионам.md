# Анализ распространения рыб по регионам России

**Дата**: 2025-02-17
**Статус**: Исследовательский документ
**Цель**: Определить `region_ids` для каждого вида рыбы

---

## Методология

**Классификация распространения**:
- **Повсеместное** (`{}`) - водится почти везде в европейской части + Урал + юг Сибири
- **Ограниченное** (список регионов) - водится только в определенных регионах
- **Северное** - только северные регионы
- **Южное** - только южные регионы
- **Сибирское/Дальневосточное** - Сибирь и Дальний Восток

**Группы регионов**:

| Группа | Коды регионов |
|--------|---------------|
| **Центральная Россия** | MOW, MOS, NIZ, VLA, IVL, KOS, YAR, TVE, SMO, KLU, TUL, RYA, ORL, LIP, TAM, KRS, BEL, BRY |
| **Северо-Запад** | SPE, LEN, PSK, NGR, KGD, ARK, VLG, KR, KO, MUR, NEN |
| **Юг России** | KDA, CR, SEV, AD, ROS, VGG, AST, KL, STA, DA, IN, KB, KC, SE, CE |
| **Поволжье** | TAT, BA, UD, CU, ME, MO, PNZ, SAR, ULY, SAM, NIZ |
| **Урал** | SVE, CHE, KGN, PER, KIR, TYU, KHM, YAN |
| **Сибирь** | OMS, TOM, NVS, KEM, ALT, AL, KK, KYA, TAY, EVK, IRK, UOB, BU, ZAB |
| **Дальний Восток** | SA, CHU, MAG, KAM, SAK, PRI, KHA, AMU, YEV |

## Детальный анализ по видам рыб

---

### 1. Щука (Esox lucius)

**Категория**: Хищная

**Распространение**: Повсеместное

**Ареал в России**:
- Водится практически во всех пресных водоемах России
- От европейской части до Дальнего Востока
- Нет только в высокогорных озерах и соленых водоемах

**Особенности**:
- Выносит кислую воду (pH 4.75)
- Выживает в заморных водоемах лучше других видов

**Регион IDs**: `{}` (пустой массив = повсеместно)

**Исключения** (где редко или нет):
- Нет точных исключений, водится везде

---

### 2. Судак (Sander lucioperca)

**Категория**: Хищная

**Распространение**: Почти повсеместное, кроме крайнего севера

**Ареал в России**:
- Европейская часть России
- Бассейны Каспийского, Азовского, Черного, Балтийского морей
- Акклиматизирован в Сибири (Обь, Иртыш, Енисей)

**Не водится**:
- Крайний Север (Мурманская область, Ненецкий АО, Ямал)
- Республика Саха (Якутия)
- Чукотка
- Камчатка
- Магаданская область

**Регион IDs**: `{}` (повсеместно в основных регионах рыбалки)

**Примечание**: В северных регионах встречается редко, но для MVP можно показывать везде

---

### 3. Окунь (Perca fluviatilis)

**Категория**: Хищная

**Распространение**: Повсеместное

**Ареал в России**:
- Самая распространенная хищная рыба

- От Карелии до Камчатки

**Регион IDs**: `{}` (повсеместно)

### 4. Жерех (Aspius aspius)

**Распространение**: Ограниченное (бассейны южных морей)

**Ареал в России**:
- Бассейн Каспийского моря (Волга, Урал)
- Бассейн Азовского моря (Дон)
- Бассейн Черного моря (Кубань)
- Редко в Балтийском бассейне

**Основные регионы**:
```sql

    'TAT', 'BA', 'SAR', 'SAM', 'ULY', 'PNZ'
```

**Не водится**:
- Сибирь (кроме юга)
- Дальний Восток
- Крайний Север

**Регион IDs**: Список из ~30 регионов

2. **Список регионов** = ограниченное распространение, показывать только в указанных регионах

3. **Источники данных**:
   - Википедия (ареалы обитания)
   - Ихтиологические исследования
   - Рыболовные базы данных
   - Красная книга РФ

4. **Дисклеймер для пользователя**:
   > "Данные о распространении рыб носят справочный характер. Ареал обитания может меняться в зависимости от сезона, погодных условий и других факторов."

5. **Обновление данных**:
   - При добавлении новых видов рыб - проводить аналогичный анализ
   - Уточнять распространение на основе обратной связи от пользователей
